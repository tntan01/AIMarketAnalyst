PROMPT 4 — PHASE 10.4: THÊM HÀM TÍNH THỐNG KÊ WIN RATE, AVG WIN, AVG LOSS, EXPECTANCY

Bạn đang làm trong project ai-market-analyst-v2.0 trên VSCode.

Bối cảnh:
Phase 10 cần tính thống kê cơ bản từ danh sách result_r đã làm sạch. Prompt này chỉ tính stats, chưa map sang evidence_score.

Mục tiêu riêng của prompt này:
Thêm hàm calculate_trade_stats(results_r) để tính thống kê lõi.

Yêu cầu rất quan trọng:
- Chỉ được sửa:
  1. core/statistical_edge_engine.py
  2. tests/test_statistical_edge_engine_stats_phase10.py
- Không sửa file khác.
- Không tích hợp vào analysis_engine.
- Không sửa journal_service.
- Không sửa UI.
- Không làm final_score.

Việc cần làm:
1. Trong core/statistical_edge_engine.py, thêm hàm:

   def calculate_trade_stats(results_r: list[float]) -> dict:
       ...

2. Logic:
   - sample_size = len(results_r)
   - wins = result_r > 0
   - losses = result_r <= 0
   - win_rate = len(wins) / sample_size nếu sample_size > 0, nếu không thì None
   - avg_win_r = trung bình wins nếu có, nếu không thì 0.0
   - avg_loss_r = abs(trung bình losses) nếu có, nếu không thì 0.0
   - expectancy_r = win_rate * avg_win_r - (1 - win_rate) * avg_loss_r
   - Nếu sample_size = 0 thì expectancy_r = None

3. Output bắt buộc:

   {
       "sample_size": ...,
       "win_count": ...,
       "loss_count": ...,
       "win_rate": ...,
       "avg_win_r": ...,
       "avg_loss_r": ...,
       "expectancy_r": ...,
   }

4. Lưu ý:
   - Breakeven 0R tính vào loss_count theo công thức losses <= 0 để bảo thủ.
   - Không làm tròn quá sớm. Nếu cần làm tròn thì chỉ làm ở UI sau này, không làm trong engine.
   - Không dùng pandas/numpy để tránh thêm dependency.

5. Tạo test mới:
   tests/test_statistical_edge_engine_stats_phase10.py

6. Test dữ liệu giống thật:
   Dataset 1:
   results = [1.5, -1.0, 2.0, -1.0, 0.5, -0.5, 0.0, 1.0, -1.0, 1.2]
   Kiểm tra:
   - sample_size = 10
   - win_count = 5
   - loss_count = 5
   - win_rate = 0.5
   - avg_win_r xấp xỉ (1.5 + 2.0 + 0.5 + 1.0 + 1.2) / 5
   - avg_loss_r xấp xỉ abs((-1.0 -1.0 -0.5 + 0.0 -1.0) / 5)
   - expectancy_r đúng theo công thức

   Dataset 2:
   results = []
   Kiểm tra không crash, sample_size = 0, expectancy_r = None.

Test bắt buộc:
- Chạy:
  python -m pytest tests/test_statistical_edge_engine_stats_phase10.py -q
- Chạy lại toàn bộ test Phase 10 đã có:
  python -m pytest tests/test_statistical_edge_engine_import_phase10.py tests/test_statistical_edge_engine_cleaning_phase10.py tests/test_statistical_edge_engine_stats_phase10.py -q
