Module này cung cấp hàm chính:

```python
def check_trade_gates(context: dict) -> dict:
    ...
```

Output bắt buộc theo format:

```python
{
    "allowed": True,
    "decision_cap": None,
    "block_codes": [],
    "warning_codes": [],
    "reasons": [],
}
```

Ý nghĩa:
- `allowed`: True/False, False nghĩa là bị chặn giao dịch.
- `decision_cap`: None hoặc một trong các giá trị:
  - `"TRADE_BLOCKED"`
  - `"WATCH_ONLY"`
  - `"WAITING_CONFIRMATION"`
- `block_codes`: list mã chặn cứng.
- `warning_codes`: list mã cảnh báo/cap mềm.
- `reasons`: list giải thích tiếng Việt ngắn gọn cho UI/log.

```python
from datetime import datetime, timedelta, timezone
from core.market_models import Candle

def make_candles(count=100, start_price=1.0800, step=0.0003):
    candles = []
    t = datetime(2026, 1, 1, tzinfo=timezone.utc)
    price = start_price

    for i in range(count):
        open_ = price
        close = price + step
        high = max(open_, close) + 0.0002
        low = min(open_, close) - 0.0002

        candles.append(Candle(
            time=t + timedelta(hours=i),
            open=open_,
            high=high,
            low=low,
            close=close,
            volume=1000,
        ))

        price = close

    return candles
```
