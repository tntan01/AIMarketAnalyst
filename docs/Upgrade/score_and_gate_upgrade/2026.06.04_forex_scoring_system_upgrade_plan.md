FOREX SCORING SYSTEM UPGRADE IMPLEMENTATION PLAN
=================================================

KẾ HOẠCH CHI TIẾT NÂNG CẤP CÁCH CHẤM ĐIỂM TRONG HỆ THỐNG PHẦN MỀM GIAO DỊCH FOREX

Mục tiêu cuối cùng:

Không để score tự quyết định vào lệnh.
Score chỉ dùng để xếp hạng setup.
Quyền vào lệnh phải do Gate kiểm soát.

=================================================
TỔNG QUAN HƯỚNG NÂNG CẤP
=================================================

Hiện tại hệ thống đang thiên về:

Trend + Momentum + Location + SMC + Risk + Macro = total score

Cần nâng thành:

Signal Score
+ Evidence Score
+ Execution Quality Score
+ Account Guard / Gate
= Final Decision

Trong đó:

Signal Score = điểm tín hiệu kỹ thuật hiện tại.
Evidence Score = điểm lợi thế thống kê từ lịch sử.
Execution Quality Score = điểm chất lượng thực thi.
Gate = cổng kiểm soát, có quyền chặn giao dịch.

Bước 6.2 — Thêm hàm expected effective R:R

def calculate_expected_effective_rr(
    direction: str,
    entry: float,
    stop_loss: float,
    take_profit: float,
    spread_price: float,
) -> float:
    if direction == "buy":
        risk = abs(entry - stop_loss)
        reward = abs(take_profit - entry)
    else:
        risk = abs(stop_loss - entry)
        reward = abs(entry - take_profit)

    effective_risk = risk + spread_price
    effective_reward = reward - spread_price

    if effective_risk <= 0:
        return 0.0

    return max(effective_reward / effective_risk, 0.0)

def calculate_edge(trades: list[dict]) -> dict:
    sample_size = len(trades)

    if sample_size < 30:
        return {
            "sample_size": sample_size,
            "evidence_score": 50,
            "confidence": "low",
            "reason_code": "STAT_EDGE_NOT_ENOUGH_DATA",
        }

    wins = [t["result_r"] for t in trades if t["result_r"] > 0]
    losses = [t["result_r"] for t in trades if t["result_r"] <= 0]

    win_rate = len(wins) / sample_size
    avg_win = sum(wins) / len(wins) if wins else 0
    avg_loss = abs(sum(losses) / len(losses)) if losses else 0

    expectancy = win_rate * avg_win - (1 - win_rate) * avg_loss

    evidence_score = map_expectancy_to_score(expectancy, sample_size)

    return {
        "sample_size": sample_size,
        "win_rate": win_rate,
        "avg_win_r": avg_win,
        "avg_loss_r": avg_loss,
        "expectancy_r": expectancy,
        "evidence_score": evidence_score,
    }

Fallback logic:

if sample_size(symbol + direction + regime) >= 50:
    use_group("symbol_direction_regime")
elif sample_size(symbol + direction) >= 50:
    use_group("symbol_direction")
else:
    evidence_score = 50

Dịch:

Nếu chưa đủ dữ liệu thì evidence score trung lập, không kết luận tốt/xấu.

PHASE 14 — DECISION ENGINE MỚI
=================================================

Mục tiêu:

Quyết định cuối cùng không nằm rải rác trong nhiều file.

Module mới:

core/decision_engine.py

Input:

{
    "signal_score": ...,
    "evidence_score": ...,
    "execution_quality_score": ...,
    "gate_result": ...,
    "entry_status": ...,
    "score_gap": ...,
}

Output:

{
    "decision": "READY_TO_TRADE",
    "final_score": 82,
    "reason_codes": [],
    "warning_codes": [],
    "block_codes": [],
}

Decision states:

READY_TO_TRADE = có thể cân nhắc vào lệnh
WAITING_CONFIRMATION = chờ xác nhận thêm
AGGRESSIVE_SETUP = setup mạo hiểm
WATCH_ONLY = chỉ theo dõi
TRADE_BLOCKED = bị chặn giao dịch
STAND_ASIDE = đứng ngoài

Logic gợi ý:

def make_final_decision(final_score, gate_result, entry_status):
    if not gate_result["allowed"]:
        return "TRADE_BLOCKED"

    if gate_result.get("decision_cap") == "WATCH_ONLY":
        return "WATCH_ONLY"

    if gate_result.get("decision_cap") == "WAITING_CONFIRMATION":
        return "WAITING_CONFIRMATION"

    if entry_status != "confirmed_entry":
        return "WAITING_CONFIRMATION"

    if final_score >= 80:
        return "READY_TO_TRADE"

    if final_score >= 65:
        return "WATCH_ONLY"

    if final_score >= 50:
        return "WAITING_CONFIRMATION"

    return "STAND_ASIDE"
