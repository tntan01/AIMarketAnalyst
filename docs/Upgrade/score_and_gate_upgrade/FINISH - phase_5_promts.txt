Bối cảnh hiện tại:
- Phase 1 đã xong: score đã có `signal_score`, có thể vẫn giữ `total` để backward compatibility.
- Phase 2 đã xong: đã có `core/trade_gate_engine.py`, output `trade_gate`, gate có quyền block/cap.

- Setup đẹp kiểu liquidity sweep + displacement + M15 strict đáng được đánh dấu chất lượng cao.

Bước 3 — Logic nhận diện cơ bản:
Vì cấu trúc SMC thật có thể khác nhau, phải đọc code hiện tại để map đúng field.

Gợi ý logic, nhưng phải điều chỉnh theo code thật:

1. Zone broken:
- Nếu `selected_zone.get("broken") is True`
- hoặc `selected_zone.get("status") == "broken"`
- hoặc zone liên quan có field `is_broken`
thì:
```python
zone_broken = True
```

2. CHOCH ngược hướng:
- Nếu direction là buy mà SMC có CHOCH bearish/down/sell thì True.
- Nếu direction là sell mà SMC có CHOCH bullish/up/buy thì True.

3. Liquidity sweep aligned:
- Nếu direction buy và có sweep đáy/sell-side liquidity rồi phản ứng tăng thì aligned.
- Nếu direction sell và có sweep đỉnh/buy-side liquidity rồi phản ứng giảm thì aligned.
- Nếu code hiện tại chỉ có field đơn giản như `liquidity_sweep_direction`, map theo field đó.

4. Displacement aligned:
- Nếu direction buy và displacement direction bullish/up/buy thì True.
- Nếu direction sell và displacement direction bearish/down/sell thì True.

from core.market_models import Candle

def make_sweep_then_displacement_buy_candles(count=120, base_price=1.0800):
    candles = []
    t = datetime(2026, 1, 1, tzinfo=timezone.utc)
    price = base_price

    for i in range(count):
        if i == 80:
            # quét đáy rồi đóng lên
            open_ = price
            low = price - 0.0020
            close = price + 0.0010
            high = close + 0.0005
        elif i > 80:
            open_ = price
            close = price + 0.00035
            high = close + 0.0002
            low = open_ - 0.00015
        else:
            open_ = price
            close = price + (0.00005 if i % 2 == 0 else -0.00003)
            high = max(open_, close) + 0.0002
            low = min(open_, close) - 0.0002

        candles.append(Candle(
            time=t + timedelta(hours=i),
            open=open_,
            high=high,
            low=low,
            close=close,
            volume=1000 + i,
        ))
        price = close

    return candles
