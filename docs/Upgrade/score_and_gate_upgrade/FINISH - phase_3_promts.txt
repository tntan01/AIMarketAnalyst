Mục tiêu:
Tạo helper tính direction bias và score_gap dựa trên điểm Buy/Sell.

Ưu tiên thêm vào file:
`core/signal_engine.py`

Nhưng phải làm theo cách ít ảnh hưởng nhất:
- Không đổi hàm scoring cũ nếu không cần.
- Không làm thay đổi output cũ của `score_scenario()` ngoài những gì Phase 1 đã làm.
- Chỉ thêm helper mới.

Thêm hàm mới:

```python
def calculate_direction_bias(buy_result: dict, sell_result: dict, min_gap: float = 10.0) -> dict:
    ...
```

Hàm này nhận:
- `buy_result`: dict điểm của BUY, ưu tiên lấy `signal_score`, fallback sang `total`.
- `sell_result`: dict điểm của SELL, ưu tiên lấy `signal_score`, fallback sang `total`.
- `min_gap`: ngưỡng tối thiểu để coi hướng là rõ ràng, mặc định 10.0.

Output bắt buộc:

```python
{
    "best_side": "buy" hoặc "sell" hoặc "neutral",
    "buy_score": float,
    "sell_score": float,
    "score_gap": float,
    "is_clear_bias": bool,
    "min_gap": float,
}
```

Logic yêu cầu:

1. Lấy điểm an toàn:

```python
buy_score = float(buy_result.get("signal_score", buy_result.get("total", 0)) or 0)
sell_score = float(sell_result.get("signal_score", sell_result.get("total", 0)) or 0)
```

2. Tính gap:

```python
score_gap = abs(buy_score - sell_score)
```

3. Xác định best_side:

```python
if buy_score > sell_score:
    best_side = "buy"
elif sell_score > buy_score:
    best_side = "sell"
else:
    best_side = "neutral"
```

4. Xác định is_clear_bias:

```python
is_clear_bias = score_gap >= min_gap
```

5. Nếu hai điểm bằng nhau thì:
```python
best_side = "neutral"
is_clear_bias = False
score_gap = 0
```

Yêu cầu an toàn:
- Không được raise exception nếu input thiếu key.
- Không được crash nếu `buy_result` hoặc `sell_result` là None. Trong trường hợp đó coi như dict rỗng.
- Không thay đổi hàm `best_side()` cũ nếu app đang dùng, trừ khi thật sự cần. Nếu cần thì chỉ sửa tối thiểu và phải giữ backward compatibility.

```python
from datetime import datetime, timedelta, timezone
from core.market_models import Candle

def make_trending_candles(count=120, start_price=1.0800, step=0.00025):
    candles = []
    t = datetime(2026, 1, 1, tzinfo=timezone.utc)
    price = start_price

    for i in range(count):
        open_ = price
        close = price + step
        high = max(open_, close) + 0.00015
        low = min(open_, close) - 0.00015
        candles.append(Candle(
            time=t + timedelta(hours=i),
            open=open_,
            high=high,
            low=low,
            close=close,
            volume=1000 + i,
        ))
        price = close

    return candles


def make_ranging_candles(count=120, base_price=1.0800):
    candles = []
    t = datetime(2026, 1, 1, tzinfo=timezone.utc)

    for i in range(count):
        offset = 0.0005 if i % 2 == 0 else -0.0005
        open_ = base_price
        close = base_price + offset
        high = max(open_, close) + 0.0002
        low = min(open_, close) - 0.0002
        candles.append(Candle(
            time=t + timedelta(hours=i),
            open=open_,
            high=high,
            low=low,
            close=close,
            volume=1000 + i,
        ))

    return candles
```
